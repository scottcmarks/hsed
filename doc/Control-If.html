(?) :: Bool -> p -> p -> p infixr 1 Source #

if' function as operator (?)

Since infixr 0 $, we can use ... ? ... $ ... like ... ?... : ... in C-family languages.

>>> 1 == 2 ? 3 $ 4
4
>>> 'a' == 'a' ? "Yes!" $ "no?"
"Yes!"
