predicate :: c a -> Either (ErrorMessage c a) (c a) Source #

Refinement predicate. Either pass a wrapped value as Right or give an error message as Left.

Note that this is passed the value after being wrapped, and therefore assumes that the wrapping operation is mostly harmless.
